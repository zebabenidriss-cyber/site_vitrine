#  NewDelice - Site Vitrine

Site vitrine multi-pages réalisé en **HTML/CSS/JS** dans le cadre d’un projet académique.  
Ce site présente le restaurant **NewDelice** : son menu, ses services, et ses coordonnées.


##  Structure du projet
 [index.html ]  → Page d’accueil avec présentation du restaurant et menu.
[services.html]   → Liste des services proposés (sur place, livraison, à emporter).
[apropos.html ]   → Histoire du restaurant, valeurs et équipe.
[contact.html]    → Formulaire de contact.
[CSS/ ]          → Dossier contenant les fichiers de style.
[images/]       → Ressources visuelles (logo, plats, photos).
[.gitignore]    → Exclusion des fichiers non nécessaires (node_modules, dist, etc.).



##  Technologies utilisées
**HTML5** : structure des pages.
**CSS3** : mise en forme et design.
**JavaScript**  : pour ajouter des interactions dynamiques.
